Pokémon API Test

Below is an example I have created showing how an API ( Application Programming Interface - Wikipedia) can be used to return data from an external source using a 'GET' method within a request. The API being used for this page is PokéAPI (https://pokeapi.co/). When this page loads, a request is sent to the API to respond with a list of Pokémon names and URLs linking to further information about the specific Pokémon. This populates the dropdown list and populates the page with images from the Pokémon with an 'id' of '1' (Bulbasaur). Any subsequent selection made from the dropdown performs another call using the 'id' number related to that selection to return the relevant images for that Pokémon.
Try it out to see!
